# CryBot-Telegram
Bot de Telegram para escaneo automático de wallets, reclamo de airdrops y venta de NFTs.

## Uso

1. Crea un archivo `.env` con tu token de bot de Telegram:

   ```
   BOT_TOKEN=TU_TOKEN_AQUI
   ```

2. Instala las dependencias y ejecuta el bot:

   ```bash
   npm install
   npm start
   ```

3. En Telegram, envía `/saldo <direccion>` para obtener el balance en ETH de la dirección indicada.
